## Changes

Major v2.0.0 release. Built on the monorepo ports-and-adapters migration + composition-root
consolidation (PRs #59–62), this release ships the user-facing v2 surface:

- **Model Browser** — searchable modal replacing the model-selection dropdown, with provider/
  family pivots and live OpenRouter metadata (pricing, context window, release date) joined onto
  the curated catalog.
- **Streaming progress stats** — chunk count, elapsed time, first-chunk latency, and average
  chunks/sec; the user-facing counter is now honestly "chunks," not "tokens."
- **Debug Output title-bar command** (`prose-minion.showOutputChannel`, `$(bug)` icon).
- **All Tools Modal copy audit** — 16 writing-tool descriptions reworded to match prompt behavior.
- **Model catalog refresh** — added GLM 5.2 + Qwen3.7 Plus, Category Search bumped to Mistral
  Large 2512, three stale models dropped; `family` field added across the curated arrays.
- **Marketplace docs** — README/CHANGELOG rewritten for the `apps/vscode-extension` package root
  (absolute GitHub URLs), FrameMinion sister-extension callout.
