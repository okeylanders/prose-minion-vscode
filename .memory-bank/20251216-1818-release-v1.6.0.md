## Changes

### N-gram Mode for Category Search
- Search for bigrams (2-word phrases) or trigrams (3-word phrases)
- Tab-based mode selector: Words / Bigrams / Trigrams
- Min occurrences filter (1-5) to reduce noise
- Mode-aware batch sizing: 400/200/150

### Category Search Cancellation
- AbortController pattern for graceful cancellation
- Red "✕ Cancel" button during search
- Preserves partial results with warning message
- Shows completed/total batches

### Punctuation Stripping
- N-grams strip punctuation for cleaner matching
- Preserves apostrophes for contractions
