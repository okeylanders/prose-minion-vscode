## Key Learning

- **Monorepo Marketplace gotcha:** vsce resolves relative README links/images against the
  repository root, but a workspace extension packages from its subdir (`apps/vscode-extension/`).
  Relative `assets/...` / `CHANGELOG.md` / `LICENSE` paths therefore 404 on the web Marketplace.
  Use absolute `raw.githubusercontent.com` (images) and `github.com/.../blob/main` (docs) URLs.
