## Dependency Audit

Initial audit notes:

- `npm audit` against the configured CodeArtifact endpoint failed with a 404 audit endpoint response.
- Reran audit against `https://registry.npmjs.org` for release verification.
- Initial public audit reported 6 vulnerabilities:
  - `@babel/core` arbitrary file read advisory.
  - `form-data` CRLF injection advisory.
  - `js-yaml` quadratic-complexity DoS advisory.
  - `linkify-it` quadratic-complexity advisory via `@vscode/vsce -> markdown-it`.

Actions taken:

- Ran non-breaking `npm audit fix --registry=https://registry.npmjs.org`.
- Upgraded `@vscode/vsce` from `^2.22.0` to `^3.9.2` to clear the remaining high advisory in the packaging toolchain.

Final audit:

- `npm audit --registry=https://registry.npmjs.org`: 0 vulnerabilities.
- `npm audit --audit-level=high --registry=https://registry.npmjs.org`: 0 vulnerabilities.
