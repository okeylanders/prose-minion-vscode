## Changes

Five new focus writing tools added to the Writing Tools Assistant, expanding the analysis suite:

- **Decision Points**: Character decision moment analysis (stakes, consequences)
- **Show & Tell**: Showing vs telling balance evaluation
- **Gestures**: Body language variation analysis
- **Choreography**: Scene-wide movement pattern analysis
- **Stock & Signature**: Cognitive economy analysis (stock vs distinctive descriptions)

Additional changes:
- Opus 4.6 model added to OpenRouter model list
- Streaming cancellation preserves partial content (endStreaming vs reset)
- Analysis UI reorganized to 2x4 grid layout
- Behavioral tests for streaming cancellation and useStreaming hook
