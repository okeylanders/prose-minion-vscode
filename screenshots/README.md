# Screenshots Organization

This folder contains screenshots for the main README and documentation.

## Folder Structure

### `/overview`
- General extension interface screenshots
- Main panel view
- Activity bar icon
- Tab navigation

### `/analysis`
- Prose Assistant examples
- Dialogue Microbeat Assistant examples
- AI-powered writing suggestions
- Before/after examples

### `/metrics`
- Prose Statistics output
- Word Frequency analysis
- Style Flags detection
- Publishing Standards comparison
- Chapter aggregation examples

### `/search`
- Word Search results
- Pattern matching examples
- Cluster detection visualization
- Context display examples

### `/dictionary`
- Word lookup results
- Craft-aware definitions
- Tonal guidance examples
- Usage examples and related terms

### `/settings`
- Settings overlay interface
- Model selection
- Publishing presets
- Context paths configuration
- Token usage widget

## Naming Convention

Use descriptive names with hyphens:
- `overview-main-panel.png`
- `analysis-dialogue-assistant.png`
- `metrics-prose-stats-output.png`
- `search-cluster-detection.png`
- `dictionary-word-lookup.png`
- `settings-overlay-full.png`

## Image Guidelines

- **Format**: PNG preferred for UI screenshots
- **Resolution**: Capture at 1x or 2x for clarity
- **Size**: Keep under 500KB when possible
- **Content**: Show realistic examples with actual writing samples
- **Privacy**: Don't include sensitive project content

## Usage in README

Reference screenshots using relative paths:
```markdown
![Prose Statistics](screenshots/metrics/prose-stats-output.png)
```

Or with titles:
```markdown
<p align="center">
  <img src="screenshots/overview/main-panel.png" alt="Main Panel" width="600"/>
  <br/>
  <em>Prose Minion's main interface with all tools accessible via tabs</em>
</p>
